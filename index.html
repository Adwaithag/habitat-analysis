Project Overview

A LiDAR-based forest analysis project focused on studying the structural characteristics of forests in Kodaikanal using handheld LiDAR data and ground-truth ecological measurements. The project includes 3D point cloud visualization, canopy height model generation, individual tree detection, GBH estimation, and comparative analysis for understanding forest structure and habitat dynamics.

Key Findings

  • Dense and multi-layered canopy structure observed across Bombay Shola.
  • LiDAR data effectively captured forest structure in 3D.
  • Canopy gaps and dense vegetation zones were clearly distinguishable.
  • Ground observations supported LiDAR-derived habitat analysis.

Conclusion

This project demonstrated the usefulness of handheld LiDAR for forest structure analysis and ecological interpretation in the shola forests of Kodaikanal. By combining LiDAR data with field observations, the study provided insights into vegetation density, canopy structure, habitat openness, and forest complexity.
